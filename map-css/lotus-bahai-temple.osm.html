Lotus Bahai Temple

Run

Colouring the Lotus Bahai Template as a lotus flower.

This example illustrates selecting the same way more than once (44827278). In the second selection we output just its centroid and then style that using way:placeholder.

We also hide nodes that are selected using the > recurse operator so they're not displayed.
